# Zahid Hussain - Portfolio

## Overview
This is the personal portfolio website of **Zahid Hussain**, a final year BCA student at BMS College of Commerce and Management specializing in web development. The portfolio showcases my projects, skills, and experience as a Frontend Engineer.

## Tech Stack
- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Icons:** Lucide React
- **Animations:** GSAP, Framer Motion

## Features
- **Responsive Design:** Optimized for all devices.
- **Performance:** Fast loading with code splitting and lazy loading.
- **Clean Architecture:** Modular component structure.
- **Modern UI:** sleek and professional design.

## Getting Started

To run this project locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/zahidb0nd/my-app.git
    cd my-app
    ```

2.  **Install dependencies:**
    ```bash
    pnpm install
    ```

3.  **Start the development server:**
    ```bash
    pnpm run dev
    ```

4.  **Build for production:**
    ```bash
    pnpm run build
    ```
